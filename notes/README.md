## Gridsomeを使った静的サイト生成

GridsomeはVue.jsで静的サイトが作れるツール
Github Pages や Netlifyなどにアップすればサーバーレスにサイトを作れる

MDやWordpressのデータをいい感じに紐付けてVueの変数に渡せる仕組みがある
